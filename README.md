# Aim of the project

The project is build to learn about Raspberry pi Pico SDK. 

# The parts used in project

- Raspberry Pi Pico RP2350 -> The main system/brain of the project
- LED -> for lighting the help signal
- Buzzer -> for warning the people
- MQ-2 -> sensor for smoke

# The working of project

The raspberry pi will first of all read the singals from the sensor which is MQ-2
sensor and also then reading from that we can use buzzer and LED to warn the people.
